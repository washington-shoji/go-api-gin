# CLI Guide

## Tailwind css

### Watch

.\tailwindcss.exe -i .\static\input.css -o .\static\output.css --watch

### Minify

.\tailwindcss.exe -i .\static\input.css -o .\static\output.css --minify

go install github.com/cosmtrek/air@latest

air init
